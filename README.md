# Disease Prediction Using Machine Learning

## 📌 Overview
This project predicts whether a patient has breast cancer using Machine Learning. The model is trained on the Breast Cancer Wisconsin dataset available in Scikit-learn.

## 🚀 Features
- Predicts breast cancer (Benign or Malignant)
- Uses the Random Forest Classifier
- Achieves 96.49% accuracy
- Simple and easy-to-understand Python implementation

## 🛠️ Technologies Used
- Python
- Scikit-learn
- NumPy

## 📊 Dataset
- Breast Cancer Wisconsin Dataset (Scikit-learn)

## 📈 Model Accuracy
- Accuracy: **96.49%**

## ▶️ How to Run
1. Clone this repository.
2. Install the required libraries:
   ```
   pip install -r requirements.txt
   ```
3. Run the program:
   ```
   python disease_prediction.py
   ```
